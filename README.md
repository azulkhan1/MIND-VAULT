# MIND VAULT
 
Mind Vault is a back-end project focused on creating a RESTful service, designed to deepen understanding and practical application of REST API principles. It demonstrates the creation and manipulation of articles through a RESTful API, showcasing the practical application of REST API principles and Database integration.

# Key Features

RESTful API: Facilitates CRUD (Create, Read, Update, Delete) operations on articles.

MongoDB Integration: Uses MongoDB for storing and retrieving article data, demonstrating effective data handling and modeling.

Express.js Framework: Employs Express.js to handle HTTP requests and responses.

# API Endpoints

GET /articles: Fetch all articles.

POST /articles: Create a new article.

DELETE /articles: Delete all articles.

GET /articles/:searchByTitle: Fetch a specific article by title.

PUT /articles/:searchByTitle: Update an article by replacing it entirely.

PATCH /articles/:searchByTitle: Partially update an article.

DELETE /articles/:searchByTitle: Delete a specific article by title.

# Setup and Installation

Clone the repository to your local machine.

Install dependencies using npm install.

Ensure MongoDB is running on your machine.

Start the server with node app.js.

Access the application through http://localhost:3000/.
